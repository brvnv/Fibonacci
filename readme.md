--------

Programa para imprimir sequência de Fibonacci (talvez você não tenha desconfiado).

O cálculo é feito através de um loop. f1 é a variável que guarda o número corrente na sequência, f2 guarda a diferença entre o número corrente e o próximo.

O número corrente é obtido somando o número anterior (inicialmente 0) e a diferença(calculada subtraindo o valor do número corrente pela diferença anterior, inicialmente 1).
